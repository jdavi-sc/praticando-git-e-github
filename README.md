# Versionamento e Colaboração com Git e GitHub

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de versionamento de código, gerenciamento de branches, merges e utilização do GitHub no fluxo de desenvolvimento.

A proposta do projeto é simular um ambiente real de desenvolvimento, permitindo a prática de organização de commits, controle de versões e colaboração utilizando ferramentas amplamente adotadas no mercado.
